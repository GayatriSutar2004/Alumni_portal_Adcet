# 🎓 Alumni Portal – Full Stack Web Application

A full-stack Alumni Portal developed using the MERN stack that enables alumni and students to connect, share information, and manage alumni records efficiently. This project demonstrates CRUD operations, REST API integration, authentication, and responsive UI design.

---

## 🚀 Features

- User Registration and Login
- Add Alumni Details
- Update Alumni Information
- Delete Alumni Records
- View Alumni List
- Search and Filter Alumni
- RESTful API Integration
- Responsive User Interface
